|discount	
|O
|Numeric
|18,3

|[[#KlarnaV2_TransactionTypes_authorization_Fields_orderItem_type]]type
|O 
|String
|n/a
a|Order item type. +
Possible values: 

  - ``shipment_fee``
  - ``handling_fee``
  - ``discount``
  - ``physical``
  - ``sales_tax``
  - [[KlarnaV2_TransactionTypes_authorization_Fields_orderItem_type_digital]]``digital``
  - ``gift_card``
  - ``store_credit``

//-

If not specified, ``type`` is treated as a regular article.
